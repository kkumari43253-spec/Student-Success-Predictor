# 🎓 Student Success Predictor

An AI-based Student Performance Prediction System that predicts a student's final score using Machine Learning.

## 📌 Project Overview

This project uses student academic information like study hours, attendance, previous marks, and assignment performance to predict the final score.

The system provides:
- Predicted Final Score
- Grade Prediction
- Performance Level
- Success Probability
- Risk Analysis
- AI-based Suggestions


## 🚀 Features

✅ Student performance prediction  
✅ Machine Learning based model  
✅ Web-based interface  
✅ Real-time prediction  
✅ Performance analysis  
✅ User-friendly UI

## 🤖 Machine Learning Model

Model Used:

**Random Forest Regressor**

Dataset:
- 10,000 student records

Input Features:
- Study Hours
- Attendance
- Previous Marks
- Assignment Score

Target:
- Final Score


## 📊 Model Performance

Evaluation Results:

- MAE: 2.67
- RMSE: 3.33
- R² Score: 0.8933
